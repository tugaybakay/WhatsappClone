## Includes
- Combine Framework
- Firebase Auth + Database + FireStore
- Custom Modifiers
- Animations
- Image picker
- MVVM Design Pattern
- CoreData
- SwiftUI

## Usage

To run the app, follow these steps:

1. Clone the repository.
2. Open the project in Xcode.
3. Build and run the app on the simulator or a physical device.
